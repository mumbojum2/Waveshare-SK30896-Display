# Waveshare 3.5" Capacitive Touch LCD for Raspberry Pi 5 (SKU 29318/30896)

Waveshare 3.5" capacitive touch LCD on Raspberry Pi 5 with Raspberry Pi OS 64-bit Lite (Bookworm/Trixie).

## Install

```bash
sudo apt update && sudo apt install git -y
git clone https://github.com/mumbojum2/Waveshare-SK30896-Display.git
cd Waveshare-SK30896-Display && chmod +x installfromdbto.sh && ./installfromdbto.sh
```

## Install From Source
```bash
sudo apt update && sudo apt install git -y

cd Waveshare-SK30896-Display && chmod +x installfromsource.sh && ./installfromsource.sh
```
